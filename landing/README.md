# SakBol Landing (Astro)

Статический сайт: главная, политика, соглашение, поддержка.

## Локально

```bash
npm install
npm run dev          # http://localhost:4321
```

## Сборка

```bash
npm run build        # → dist/
```

В production содержимое `dist/` копируется в `/var/www/sakbol-landing` (nginx).
