# Setup: Google Gemini (AI)

Заменяем DeepSeek / Groq на Google Gemini. Один провайдер для:
- Распознавания PDF/фото анализов (vision)
- Разбора и перевода медицинских документов
- ИИ-чата с пользователем

## Модель

Бэкенд читает имя модели из ENV — без правки кода.

```env
GEMINI_MODEL=gemini-2.5-pro
```

| Модель | Когда использовать |
|--------|--------------------|
| `gemini-2.5-pro` | Стабильная, сейчас по умолчанию. Точная, дороже, медленнее. |
| `gemini-2.5-flash` | Быстрее и дешевле — для чата и коротких ответов. |
| `gemini-3.1-pro` *(когда станет доступна)* | Просто поменять значение `GEMINI_MODEL` — деплой не нужен, перезапуск контейнера и всё. |

Доступные модели и тарифы: https://ai.google.dev/gemini-api/docs/models

## 1. Получить API-ключ

1. Откройте https://aistudio.google.com/apikey
2. Войдите аккаунтом Google.
3. **Create API key** → выберите проект Google Cloud (либо создайте новый «sakbol»).
4. Скопируйте ключ.

> Бесплатный тариф: ~15 запросов/мин на `gemini-2.5-flash`, лимит общий на проект. Для прода рекомендуется включить billing в Google Cloud (Gemini подключается к проекту GCP).

## 2. Подставить в backend `.env`

```env
GEMINI_API_KEY=AIzaSy...
GEMINI_MODEL=gemini-2.5-pro
GEMINI_TIMEOUT_MS=60000
```

## 3. Безопасность / Privacy

- В политике конфиденциальности (`landing/src/pages/privacy.astro`) указано, что текст и изображения передаются Google. Без явного действия пользователя ИИ-разбор не запускается.
- Перед отправкой документа на Gemini мы анонимизируем ФИО и ПИН (модуль `pii-scrubber`, перенесём из старого проекта).
- Запросы идут с серверной стороны — API-ключ никогда не попадает в Flutter/Web.

## 4. Тарификация

На момент запуска (май 2026):
- `gemini-2.5-pro`: ~$1.25 / 1M input tokens, $5 / 1M output (примерно).
- `gemini-2.5-flash`: в 5–10 раз дешевле.

Один разбор анализа ≈ 5–15 тыс. токенов = $0.01–0.05 на Pro, $0.001–0.005 на Flash.

Для прода: включить мониторинг расходов в Google Cloud + лимит трат (например, $50/мес — алёрт).

## 5. До получения ключа

В development без `GEMINI_API_KEY` работает `StubAiProvider`:
- Возвращает заранее заданный мок-результат OCR.
- Чат отвечает шаблоном «AI временно недоступен».

Это позволяет писать и тестировать UI Flutter без живых вызовов API.
